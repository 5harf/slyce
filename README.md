## Tech Stack

Node.js with Express

PostgreSQL with Knex

## To Run Locally

###PostgresSql 

set up PostgreSQL database named slyce

###From within the root directory:

```sh
npm install
```
Populate database:

```sh
npm run dbup
```

Start server:

```sh
node index.js
```

Navigate to localhost:8080 to see qa-worktest.html instruction page
